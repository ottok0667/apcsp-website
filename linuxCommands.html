Linux Commands

  • ls: lists directory contents

    • $cd scripts -- opens the scripts folder

    $ls -- lists the contents of the scripts folder

  • cd: change directory

    • if currently located in scripts folder

    $cd -- returns user to the home directory

  • pwd: print working directory

    • $cd scripts -- moves user from the home directory to the scripts directory

    $pwd

    /home/kelsi/scripts

  • mkdir: make new directory

    • $mkdir website

    *creates a directory by the name of "website" in the current directory*

  • cp: copy files/directories

    • $cp /var/www/html/kelsi/index.html .

    *moves a copy of the file "index.html" to the current directory from the specified directory*

  • mv: move files/directories

    • $mv /var/www/html/kelsi/index.html .

    *moves file "index.html" to the current directory from the specified directory*

  • rm: remove files/directories

    • $rm /var/www/html/kelsi/index.html

    *removes file "index.html" from the specified directory*

  • nano: creates a file of the specified name or opens the file of that name from the current directory

    • $nano index.html

    *opens the file "index.html" if it exists in the current directory; otherwise, a file by the name "index.html" is created and opened*

  • echo: prints the string provided

    • $echo hello world

    hello world

  • hostname: prints the name of the raspberry pi that you are connected to

    • $hostname

    rpi06

  • man: provides the manual of the specified command

    • $man ls

    *prints user commands of the ls function*

  • history: prints the history of all commands ever typed on that machine

    • $history

    1 ls

    2 cd

    3 cd ..

    ...

  • date: provides the current date and time

    • $date

    Tue 15 May 19:00:35 UTC 2018
